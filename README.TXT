File SMS Gateway / SMS Server.
	- Membaca SMS pada Inbox (inbox.php)
	- Membaca SMS yang terkirim pada Outbox (outbox.php)
	- Membaca SMS pada Sentitems  (sentitem.php)
	- Pengiriman SMS dari browser (kirimsms.php)
	- Pendaftaran member lewat sms (members.php)
	- Koneksi ke database MySQL lewat mysqli (mysql_koneksi.php)
	- Fungsi INSERT, DELET,UPDATE, SELECT MySQL (class_query.php)
	- Auto reply / Auto proses SMS yang masuk (runonreceive.php)
	- Link Menu (menu.php) 
	
File runonreceive.php di jalankan lewat runonreceive pada gammu-smsdrc
